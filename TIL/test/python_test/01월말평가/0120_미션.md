# 1.2 자릿수 더하기

자연수 number를 입력 받아, 자릿수의 합을 계산하여 출력하시오.

```python
sum_of_digit(1234) #=> 10
sum_of_digit(4321) #=> 10
```

---

```python
def sum_of_digit(nums):
    x = list(str(nums))
    res = 0
    for i in range(len(x)):
        res += int(x[i])
    return res

print(sum_of_digit(1234))
```
